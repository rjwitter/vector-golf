# Vector Golf

This is a game to learn and practice working with vectors.  In this version of golf, strokes are taken by specifying the angle and magnitude.

This entire game was written by Google Antigravity with no manual coding.

### What I set out to explore or build
I had this idea for this educational game years ago, but never had the time or interest to implement it. This was a chance to see how good AI development is today and see if my idea was worthwhile both as a educational tool and a fun way to kill time.

### How I approached it
I started with a simple one hole, one player game giving the basic rules.  For the one-hole version, I added obstacles, out-of-bounds rules, and a scorecard.
Once the one-hole version was reasonable, I had it expand to 9 holes and add a splash page so I could select holes to test.  Some holes had to be redesigned to be playable.  Later, the multi-player version was added.

### What I learned along the way
- AI can go a long way in writing the code.
- Be exact in your prompting.  If there is ambiguity in your prompting, the AI may go in the wrong direction.
- Be thorough in your prompting.  Try to specify as much at once as possible.  The AI seemed better at creating than refining.
